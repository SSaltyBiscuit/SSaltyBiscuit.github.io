REPO: https://github.com/kajm1ller/BG_Hackathon

This is the project I made for the 2024 BGSU Hackathon with my friend Kaj Miller that won us first place.

This project was made using JavaScript and Node.js to make a game that would allow users to make an account and compete with others to earn points by recycling. This uses a novel combination of a web based scanner and AI analysis to educate users and score them.

Watch the youtube video to see the project in action.
